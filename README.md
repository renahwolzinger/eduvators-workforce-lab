# Workshift

Workshift is an interactive research dashboard for exploring how AI may affect the US job market. It combines sourced national benchmarks with a clearly labeled occupation and scenario model.

## Features

- National AI exposure, adoption, sentiment, and employment benchmarks
- Searchable occupation explorer with sector and exposure filters
- Role-level task shifts, employment context, and resilient skills
- Curriculum Studio with eight college-program pathways, skill targets, assessment patterns, and implementation plans
- Program Gap Analyzer for screening pasted course catalogs across five readiness competencies
- Persistent saved-role state and light/dark themes
- Interactive 2030 scenario model for adoption, augmentation, and reinvestment
- Responsive desktop and mobile layouts

## Run

Open `index.html` directly in a browser. There is no build step or package dependency.

## Data note

Headline benchmarks link to Pew Research Center, the US Bureau of Labor Statistics, and O*NET. Occupation exposure scores and scenario outputs are illustrative Workshift models, not official forecasts.
